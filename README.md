Настройка нового vps-сервера практически у любого хостер-провайдера:

  - Добавление нового пользователя
  - Блокировка root: вход по паролю, вход по tty|ssh
  - Смена порта SSH: 22 -> 32755
  - Вход для нового пользователя только по SSH через SSH_AUTH_KEY
  - Настройка ufw для SSH (для любого ip или список ip для SSH входа)
  - Настройка SSH для github.com для нового пользователя
  - Настройка sysctl для оптимальной работы vps как прокси или VPN сервера

Директории проекта:

  ./data/  - Настройки ssh конфига, убрано все закомментированные настройки по дефолту, изменены следующие параметры: 
            блок входа для root, активирован вход по ssh public key

  ./monitoring/  - Скрипты мониторинга системы, копируются в домашнюю директорию нового пользователя

Шаги по установке:

1: Скопировать шаблон env в рабочий файл настроек ".env" 
cp "env_markup" ".env"

2: Запустить скрипт настройки сервера
./main.sh
